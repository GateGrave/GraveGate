3) Runtime Layer
Implemented:
- apps/runtime/src/eventBus.js
- apps/runtime/src/orchestrator.js
- apps/runtime/src/gameLoopHarness.js

Behavior in place:
- Event bus: subscribe(event_type, handler), publish(event)
- Publish executes all handlers for event_type
- Handlers can emit events; emitted events are collected
- Orchestrator processes queue of events until empty
- Orchestrator uses max_events guard for loop safety

Runtime tests:
- apps/runtime/src/testing/eventBus.test.js
- apps/runtime/src/testing/orchestrator.test.js
